## 📖 About the Series

The **Liberia Core Mathematics A+ Series** is a comprehensive, nationally aligned mathematics textbook series authored by **Sir Sammy Mohammed Sackor** (BSc Mathematics, MBA in AI & Data Science) and a team of expert Liberian educators.

The series bridges the gap between classroom theory and WASSCE examination success — using real Liberian contexts, step-by-step pedagogical scaffolding, and the proven **Rule of Three** teaching method (verbal, numerical, and algebraic explanation of every concept).

## 🛠️ Technical — About This Repository

This repository contains the official **LMC-Series website and mobile-optimized web app** — a single self-contained HTML file that works on:

- 📱 Mobile phones (iOS & Android)
- 💻 Desktop computers
- 📟 Tablets

The site is hosted on **GitHub Pages** and requires no server, database, or backend — it runs entirely in the browser with all images embedded.

**Built with:** HTML5 · CSS3 · Vanilla JavaScript · Google Fonts (Playfair Display + Plus Jakarta Sans)
